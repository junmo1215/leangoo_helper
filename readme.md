# leangoo_helper
leangoo的一些扩展功能
使用python编写的一个控制台程序

使用的是命令 + 参数的模式
使用方法：
    *`<command> [options]`*


tips：
- 在使用过程中可以使用help查看帮助
- 使用exit可以退出程序
- 使用命令名称 + -h选项可以获取对应指令的说明

#### **个人建议在使用之前先对当前看板做一个备份,以免程序有bug造成不必要的影响**
####**执行的命令和参数均不能包含空格，无论这个空格是否在括号或引号中**
